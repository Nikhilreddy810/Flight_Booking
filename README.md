# ✈ Flight Booking System

A Java-based Flight Booking application that allows users to search, book, and manage flights.

## 📌 Features
- Search available flights
- Book tickets
- Manage passenger details
- View booking history
- Cancel reservations

## 🛠 Tech Stack
- **Language:** Java
- **Framework:** Spring Boot
- **Database:** MySQL
- **Build Tool:** Maven

## 🚀 Getting Started

### Prerequisites
- Java 17 or higher
- Maven
- MySQL
